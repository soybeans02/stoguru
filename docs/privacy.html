プライバシーポリシー

最終更新日: 2026年4月9日

ストグル（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。本ポリシーでは、本アプリが収集する情報とその利用目的について説明します。

1. 収集する情報

  • アカウント情報: メールアドレス、ニックネーム（アカウント登録・ログインに使用）
  • 位置情報: 現在地の緯度・経度（近くのお店の表示、距離フィルターに使用。端末の許可が必要です）
  • 写真: プロフィール写真、お店の写真（ユーザーが選択した場合のみアップロード）
  • 利用データ: 保存したお店、フォロー情報、レビュー内容

2. 情報の利用目的

  • アプリの機能提供（お店の表示・保存・検索・共有）
  • ユーザー間のフォロー・ランキング機能の提供
  • アプリの改善・不具合対応

3. 情報の共有

本アプリは、ユーザーの個人情報を第三者に販売・貸与しません。以下の場合を除き、外部に共有することはありません。

  • ユーザーが公開設定にしたプロフィール・保存情報（フォロワーに表示）
  • 法令に基づく開示要求があった場合

4. データの保存

データはAmazon Web Services（AWS）上に安全に保存されます。写真はAmazon S3に保存され、アカウント情報はAmazon DynamoDB および Amazon Cognitoで管理されます。

5. データの削除

ユーザーはアプリ内の「アカウント削除」機能からいつでもアカウントと関連データを削除できます。削除されたデータは復元できません。

6. 第三者サービス

  • Amazon Web Services: データの保存・認証
  • Apple MapKit: 地図表示・お店検索

これらのサービスにはそれぞれのプライバシーポリシーが適用されます。

7. お子様のプライバシー

本アプリは13歳未満のお子様を対象としていません。13歳未満の方の個人情報を意図的に収集することはありません。

8. ポリシーの変更

本ポリシーは必要に応じて更新されることがあります。重要な変更がある場合はアプリ内で通知します。
